# 骰子音效文件

## 需要的音效文件

1. **cup-shake.mp3** - 摇盅摩擦声（循环）
   - 时长：1-2秒
   - 类型：摩擦、滚动声
   - 音量：中等

2. **dice-collision.mp3** - 骰子碰撞声
   - 时长：0.1-0.2秒
   - 类型：清脆的碰撞声
   - 音量：较大

3. **cup-drop.mp3** - 筛盅落下声
   - 时长：0.3-0.5秒
   - 类型：重物落地声
   - 音量：大

4. **cup-lift.mp3** - 抬盅声
   - 时长：0.5-1秒
   - 类型：玻璃摩擦声
   - 音量：中等

5. **result-show.mp3** - 结果展示音效
   - 时长：1-2秒
   - 类型：胜利/揭晓音效
   - 音量：较大

## 音效来源

### 方案1：免费音效库
- Freesound.org
- Zapsplat.com
- Mixkit.co

### 方案2：AI生成
- ElevenLabs
- Soundraw

### 方案3：录制
- 使用真实骰子和玻璃杯录制

## 临时方案

如果没有音频文件，系统会：
1. 使用Web Audio API生成简单音效
2. 静音模式运行
3. 不影响动画流程

## 格式要求

- 格式：MP3 或 OGG
- 采样率：44.1kHz
- 比特率：128kbps
- 声道：单声道或立体声
